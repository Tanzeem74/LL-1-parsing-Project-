# LL(1) Parser Visualizer 🚀

A web-based tool to visualize the step-by-step process of **LL(1) Parsing**, a core concept in Compiler Design. This project helps students and developers understand how a Top-Down parser constructs FIRST/FOLLOW sets and validates input strings using a Predictive Parsing Table.

🔗 **Live Demo:** [https://tanzeem74.github.io/LL-1-parsing-Project-/](https://tanzeem74.github.io/LL-1-parsing-Project-/)

---

## ✨ Key Features

- **Grammar Input:** Users can provide their own context-free grammars (CFG).
- **FIRST & FOLLOW Calculation:** Automatically calculates the sets for all non-terminals.
- **Parsing Table:** Generates a complete LL(1) Predictive Parsing Table.
- **Step-by-Step Validation:** Shows how a string is parsed using a stack-based approach.
- **Modern UI:** Clean, responsive dashboard-style interface built with Tailwind CSS.

## 🛠 Tech Stack

- **Frontend:** HTML5, Tailwind CSS
- **Logic:** Vanilla JavaScript (ES6+)
- **Deployment:** GitHub Pages

## ⚙️ How It Works

1. **Input Grammar:** Enter production rules (e.g., `E -> T E'`).
2. **Set Construction:** The algorithm recursively computes FIRST and FOLLOW sets.
3. **Table Creation:** Based on the sets, it fills the parsing table cells.
4. **Stack Simulation:** Real-time stack tracing for input string validation.

3. Run the project:
   Simply open `index.html` in any modern web browser.
